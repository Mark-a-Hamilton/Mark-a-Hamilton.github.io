This is my first Python project which was presented to me as a technical test in a Job Application process. At that point I had never used Python before but had concentrated on 'C' type languages, more recently C#. After a bit of research I submitted the first effort which was 95% of the way there but not quite. and consisted of 4 methods, these are :-

  1. showMessage(URL) - The Entry Method Accepts the URL parameter & calls the appropriate methods in turn.
  2. getData(URL) - Gets the Data from the response of the URL request and returns the string docData.
  3. parseGoogleDocData(docData) - Creates & populates a dictionary using the coordinates as the indexes and Character as the value stored. returns message dictionary object.
  4. printSecretMessage(message) - Prints the Characters at the coordinates revealing the message in the console window.
initially I used a text editor by writing the Pseudo Code, then converting into Python syntax, but couldn't test. After a bit more research I found Replit.com and used the online interpreter to debug the code resulting in the final draft which produces the Secret Message.

Task- Create a Python application to do the following :-

  1. Parameter - Accept the URL of a google Document as the parameter.
  2. Grid - Using the URL make a Request and return the google document grid. See Grid Format below
  3. Dictionary - Store the Grid in a Dictionary object.
  4. Print Secret Message - Print the Message in the Console Window.
Grid Format :-
  1. X-Coordinate - This is the characters X-Coordinate final position.
  2. Character - This is the character to be printed.
  3. Y-Coordinate - This is the characters Y-Coordinate final position.
When the characters are printed at the correct coordinates a secret Message will be displayed in Uppercase letters.
URL - The URL is currently called in the script and can be replaced by another URL as required.

The whole point of this project is to demonstrate how by applying standard Object Oriented Programming Principles and Methodologies I am able to adapt to different languages and circumstances.
